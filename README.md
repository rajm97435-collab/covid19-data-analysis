# 🦠 COVID-19 Data Analysis

An exploratory data analysis (EDA) of global COVID-19 data using **Python** to understand the spread, death rate, recovery rate, and country-wise impact.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

---

## 📁 Dataset Overview
- **Records:** 35,156 rows
- **Columns:** 10 features
- **Source:** Global COVID-19 grouped dataset (full_grouped.csv)

---

## 🔍 What I Did

### Data Cleaning
- Confirmed **zero missing values** and **zero duplicates**
- Converted `Date` column to datetime format for time-series analysis

### Analysis & Insights
- Derived global statistics:
  - **16.4M** confirmed cases
  - **3.97%** global death rate
  - **57.45%** global recovery rate
- **USA** identified as the most affected country with **4.29M cases (26% of global total)**
- **Americas WHO region** leads with **54% share** of global cases

### Visualizations
- Daily new case trend (line chart)
- Top 10 most affected countries (bar chart)
- WHO region-wise case distribution
- Death rate and recovery rate comparison

---

## 📌 Key Findings

| Metric | Value |
